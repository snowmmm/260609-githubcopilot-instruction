# ポモドーロタイマー 段階的実装計画

## ステップ 1：プロジェクト基盤の構築
**目標：Flask が起動し、空のページが表示される状態にする**

- `requirements.txt` の作成（Flask, flask-cors）
- `app.py` に Flask アプリの最小構成を実装
- `templates/index.html` のベース HTML を作成
- `static/css/style.css` に背景色・フォントなど基本スタイルを適用
- 動作確認：`python app.py` でサーバー起動・ブラウザ表示

---

## ステップ 2：UI の静的実装
**目標：モック画像と同等の見た目を HTML/CSS だけで再現する**

- ウィンドウ風カード（白背景・角丸・影）
- SVG 円形プログレスバーの静的配置
- タイムグラフィック（「25:00」）の配置
- 「作業中」モード表示ラベル
- 「開始」「リセット」ボタンのスタイリング
- 「今日の進捗」パネル（完了数・集中時間）の静的表示
- 紫グラデーション背景

---

## ステップ 3：タイマー基本動作の実装（JS）
**目標：開始・一時停止・リセットが動作するタイマーにする**

- `timer.js` に状態管理オブジェクト `state = { running, remaining, mode }` を実装
- `setInterval` によるカウントダウン処理
- MM:SS 形式での時間表示更新
- 「開始」ボタン押下で開始 / 再押下で一時停止
- 「リセット」ボタンで初期時間（25:00）に戻す
- SVG `stroke-dashoffset` を残り時間に連動させてプログレスバーを動的更新

---

## ステップ 4：モード切り替えとサイクル管理
**目標：作業 → 短休憩 → 作業のサイクルが自動で回る**

- タイマー完了時に次のモードへ自動遷移（作業 25 分 → 短休憩 5 分）
- モードに応じて「作業中」「短休憩」表示を切り替え
- プログレスバーの色をモードで変化させる（作業：紫 / 休憩：グレー）
- ブラウザ通知 または ビープ音での完了アラート
- ブラウザタブのタイトルを残り時間で更新（例：`25:00 - 作業中`）

---

## ステップ 5：バックエンド API とデータ永続化
**目標：完了数・集中時間が SQLite に記録・表示される**

- `models/session.py` の実装（DB 初期化・セッション保存・統計取得）
- `POST /api/session/complete` エンドポイントの実装
- `GET /api/stats/today` エンドポイントの実装
- タイマー完了時に `fetch` で API を呼び出してセッションを記録
- 今日の進捗パネルをリアルタイムで更新

---

## ステップ 6：拡張機能の実装
**目標：より実用的なアプリに仕上げる**

- 4 ポモドーロごとのロングブレイク（15 分）への自動切替
- 作業時間・休憩時間のカスタマイズ設定 UI
- 過去の履歴表示（日別グラフ）

---

## 実装ロードマップ

```
ステップ1  基盤構築         ← 依存関係・起動確認
ステップ2  静的 UI          ← モック再現・スタイリング
ステップ3  タイマー動作     ← カウントダウン・JS ロジック
ステップ4  サイクル管理     ← モード遷移・通知
ステップ5  データ永続化     ← API・SQLite・進捗表示
ステップ6  拡張機能         ← ロングブレイク・設定・履歴
```

各ステップは独立して動作確認できる粒度になっており、ステップ 5 完了時点で UI モックと同等のアプリが完成します。
