# API リファレンス

ポモドーロタイマーアプリの REST API 仕様です。

ユーザー識別はクッキーベースの Flask セッション (`user_id`) により行われます。クライアントはセッションクッキーを保持する必要があります。

---

## エンドポイント一覧

| メソッド | パス                      | 説明                           |
|--------|--------------------------|-------------------------------|
| GET    | `/`                      | タイマー画面（HTML）を返す       |
| POST   | `/api/session/complete`  | ポモドーロセッションを記録する    |
| GET    | `/api/profile`           | ユーザープロフィールを取得する    |
| GET    | `/api/stats/weekly`      | 過去 7 日間の統計を取得する      |
| GET    | `/api/stats/monthly`     | 過去 4 週間の統計を取得する      |

---

## GET `/`

タイマー画面の HTML を返します。A/B テストグループ (`ab_group`) がテンプレートに渡され、ゲーミフィケーション機能の表示有無を制御します。

**クエリパラメータ（任意）**

| パラメータ | 値         | 説明                                           |
|----------|-----------|----------------------------------------------|
| `ab`     | `on`/`off` | A/B テストグループを手動でオーバーライドする（テスト用） |

---

## POST `/api/session/complete`

完了したポモドーロセッションを記録し、更新後の統計・バッジ情報を返します。

**リクエストボディ（JSON, 任意）**

```json
{
  "duration": 25
}
```

| フィールド  | 型      | デフォルト | 説明                    |
|-----------|--------|-----------|------------------------|
| `duration` | integer | `25`      | セッション時間（分単位） |

**レスポンス `200 OK`**

```json
{
  "ok": true,
  "xp_gained": 100,
  "stats": {
    "total_sessions": 5,
    "total_xp": 500,
    "level": 2,
    "xp_in_level": 300,
    "xp_needed": 300,
    "current_streak": 2,
    "max_streak": 3,
    "weekly_sessions": 5
  },
  "badges": [
    {
      "id": "first_pomodoro",
      "name": "はじめの一歩",
      "description": "初めてのポモドーロを完了した",
      "icon": "🍅",
      "earned": true
    }
  ],
  "newly_earned": [
    {
      "id": "first_pomodoro",
      "name": "はじめの一歩",
      "description": "初めてのポモドーロを完了した",
      "icon": "🍅",
      "earned": true
    }
  ]
}
```

| フィールド      | 型       | 説明                                      |
|--------------|---------|------------------------------------------|
| `ok`         | boolean | 常に `true`                               |
| `xp_gained`  | integer | 今回獲得した XP（固定値: 100）              |
| `stats`      | object  | 更新後の統計情報（後述）                    |
| `badges`     | array   | 全バッジの獲得状況一覧                      |
| `newly_earned` | array | このセッションで条件を満たしたバッジの一覧  |

---

## GET `/api/profile`

ユーザーの統計情報・バッジ・A/B テストグループを取得します。

**レスポンス `200 OK`**

```json
{
  "stats": { ... },
  "badges": [ ... ],
  "ab_group": "on"
}
```

| フィールド   | 型      | 説明                          |
|-----------|--------|------------------------------|
| `stats`   | object | 統計情報（後述）               |
| `badges`  | array  | 全バッジの獲得状況一覧          |
| `ab_group` | string | `"on"` または `"off"`         |

---

## GET `/api/stats/weekly`

過去 7 日間（今日を含む）の日別ポモドーロ完了数を返します。

**レスポンス `200 OK`**

```json
{
  "labels": ["06/03", "06/04", "06/05", "06/06", "06/07", "06/08", "06/09"],
  "data":   [2, 0, 3, 1, 4, 2, 1]
}
```

| フィールド  | 型     | 説明                               |
|-----------|-------|-----------------------------------|
| `labels`  | array | 日付ラベル（`MM/DD` 形式）、古い順    |
| `data`    | array | 各日のポモドーロ完了数               |

---

## GET `/api/stats/monthly`

過去 4 週間の週別ポモドーロ完了数を返します（各週の月曜日始まり）。

**レスポンス `200 OK`**

```json
{
  "labels": ["05/13~", "05/20~", "05/27~", "06/03~"],
  "data":   [8, 12, 5, 10]
}
```

| フィールド  | 型     | 説明                                     |
|-----------|-------|------------------------------------------|
| `labels`  | array | 週開始日ラベル（`MM/DD~` 形式）、古い順    |
| `data`    | array | 各週のポモドーロ完了数                     |

---

## 統計情報オブジェクト（`stats`）

`/api/session/complete` および `/api/profile` のレスポンスに含まれます。

| フィールド         | 型      | 説明                                   |
|-----------------|--------|----------------------------------------|
| `total_sessions` | integer | 累計ポモドーロ完了数                    |
| `total_xp`       | integer | 累計 XP（`total_sessions × 100`）       |
| `level`          | integer | 現在のレベル（1〜10）                   |
| `xp_in_level`    | integer | 現レベル内での XP 進捗                  |
| `xp_needed`      | integer | 次レベルへ必要な XP                     |
| `current_streak` | integer | 現在の連続日数（今日または昨日が最終日） |
| `max_streak`     | integer | 過去最高の連続日数                      |
| `weekly_sessions` | integer | 今週（月〜日）の完了数                  |
