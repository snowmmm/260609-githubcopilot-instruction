# フロントエンド仕様

フロントエンドは `templates/index.html` に CSS・JavaScript をインライン記述した単一 HTML ファイルです。外部依存は Chart.js 4（CDN）のみです。

---

## 全体構成

```
templates/index.html
├── <style>        CSS（CSS カスタムプロパティ + レスポンシブ）
├── HTML 構造
│   ├── #particle-canvas    パーティクルエフェクト用 Canvas
│   ├── .ab-banner          A/B テストグループ表示バナー
│   ├── .timer-card         タイマー UI
│   └── #gamificationArea   ゲーミフィケーション UI（A/B: on 時のみ表示）
│       ├── .xp-bar-wrap    XP バー
│       ├── .streak-card    ストリーク表示
│       ├── .badges-card    バッジグリッド
│       └── .charts-wrap    週間・月間チャート
└── <script>       JavaScript
```

---

## CSS デザインシステム

CSS カスタムプロパティ（変数）でデザイントークンを管理しています。

| 変数          | 値         | 用途                     |
|------------|-----------|------------------------|
| `--red`    | `#e74c3c` | タイマーリング・ボタン         |
| `--green`  | `#27ae60` | 休憩モードリング・通知         |
| `--blue`   | `#2980b9` | スキップボタン               |
| `--gold`   | `#f39c12` | XP バー・見出し             |
| `--bg`     | `#1a1a2e` | ページ背景                  |
| `--card`   | `#16213e` | カード背景                  |
| `--card2`  | `#0f3460` | セカンダリカード・バッジ背景   |
| `--text`   | `#eaeaea` | 主要テキスト                |
| `--muted`  | `#aaa`    | サブテキスト                 |
| `--radius` | `16px`    | 角丸半径                    |

ブレークポイントは `600px` で、チャートグリッドが 2 列から 1 列に切り替わります。

## JavaScript コンポーネント

### タイマー機能

**状態変数**

| 変数               | 説明                                      |
|-----------------|------------------------------------------|
| `totalSeconds`   | 現在モードの総秒数                         |
| `remainSeconds`  | 残り秒数                                  |
| `timerInterval`  | `setInterval` のハンドル                  |
| `running`        | タイマー動作中フラグ                       |
| `currentMode`    | `"work"` / `"short"` / `"long"`          |
| `particlesActive`| パーティクルエフェクト有効フラグ            |

**定数**

| 定数             | 値      | 説明                      |
|---------------|--------|--------------------------|
| `CIRCUMFERENCE` | `552.92` | SVG リング周長（2π × 88） |

**主要関数**

| 関数                   | 説明                                                        |
|---------------------|-------------------------------------------------------------|
| `setMode(mode, min)` | モード変更（タイマーリセット・タブ状態更新）                 |
| `formatTime(sec)`    | 秒数を `MM:SS` 形式に変換                                    |
| `updateDisplay()`    | タイマー表示・リング進捗・色・タイトル・エフェクト状態を更新 |
| `progressColor(ratio)` | 残り時間の比率から青→黄→赤のグラデーション色を返す         |
| `startTimer()`       | インターバル開始・ボタンラベル変更                           |
| `stopTimer()`        | インターバル停止・ボタンラベルリセット                       |
| `resetTimer()`       | 停止して残り時間をリセット                                   |
| `onTimerComplete()`  | 作業モード完了時に API を呼び出し、通知を表示                |

**タイマーモード**

| モード       | `data-mode` | 時間    |
|-----------|------------|--------|
| 作業        | `work`     | 25 分  |
| 短休憩      | `short`    | 5 分   |
| 長休憩      | `long`     | 15 分  |

### タイマーリング（SVG Progress Ring）

SVG の `<circle>` 要素の `stroke-dashoffset` を操作して進捗を表示します。

- 半径: 88px
- 周長: `2 × π × 88 ≈ 552.92px`
- リングは `-90deg` 回転し、12 時位置から始まります

**色変化（作業モードのみ）**

`progressColor(ratio)` が線形補間（`lerpColor`）を用いて色を算出します。

| ratio の範囲    | 色変化         |
|--------------|-------------|
| 1.0 → 0.5    | 青 → 黄      |
| 0.5 → 0.0    | 黄 → 赤      |

休憩モードは固定で緑（`#27ae60`）を使用します。

### 通知トースト

```javascript
showNotification(msg)
```

画面右上に 4 秒間表示されるトースト通知です。`#notification` 要素の CSS クラス `show` を付け外しすることでフェードアニメーションを実現しています。

セッション完了時にブラウザの Web Notifications API も使用します（許可が得られている場合のみ）。

### ゲーミフィケーション UI

A/B グループが `"on"` の場合のみ `#gamificationArea` を表示します（`display: block`）。

| 関数                     | 説明                                                       |
|------------------------|----------------------------------------------------------|
| `initGameUI()`          | グループ判定後に UI 表示・プロフィール読み込み・チャート初期化 |
| `loadProfile()`         | `/api/profile` を取得して UI を更新                        |
| `updateProfileUI(stats, badges)` | XP バー・ストリーク・セッション数・バッジを更新    |
| `renderBadges(badges)`  | バッジグリッドを動的に生成                                  |
| `loadWeeklyChart()`     | `/api/stats/weekly` を取得して棒グラフを描画               |
| `loadMonthlyChart()`    | `/api/stats/monthly` を取得して折れ線グラフを描画          |

**レベル名定数**

```javascript
const LEVEL_NAMES = ["","ビギナー","見習い","集中者","熟練者","達人",
                      "エキスパート","マスター","グランドマスター","レジェンド","神"];
```

### チャート

Chart.js 4 を使用します。

| チャート ID       | 種別   | データソース            | 説明              |
|----------------|------|----------------------|-----------------|
| `weeklyChart`  | 棒グラフ | `/api/stats/weekly`  | 過去 7 日間の日別数  |
| `monthlyChart` | 折れ線  | `/api/stats/monthly` | 過去 4 週間の週別数  |

### パーティクルエフェクト

`#particle-canvas`（全画面固定の `<canvas>`）上にパーティクルを描画します。作業モードのタイマー動作中（`particlesActive === true`）のみ新しいパーティクルが生成されます。

**`Particle` クラス**

| プロパティ    | 説明                        |
|-----------|---------------------------|
| `x`, `y`  | 現在座標（画面下から上へ移動） |
| `vx`, `vy`| 速度ベクトル                 |
| `r`       | 半径（1.5〜3.5px）           |
| `life`    | 現在のライフカウンタ           |
| `maxLife` | 最大ライフ（220〜420 フレーム）|
| `hue`     | 色相（200〜240、青系）        |

| 定数・設定                    | 値    |
|---------------------------|------|
| 最大同時パーティクル数          | 100  |
| 毎フレームの生成確率            | 35%  |
| 初期プール数（シード）           | 50   |

アニメーションループは `requestAnimationFrame` で動作し続けます。パーティクルが非アクティブの場合は描画をスキップします。

### リップルアニメーション

`.ripple-layer` 内の 3 つの `<div class="ripple">` が CSS アニメーション（`ripple-out`）で拡散します。各要素は `1.15s` ずつ遅延し、連続して広がるエフェクトを実現します。作業モードのタイマー動作中のみ表示されます（`display: block`）。

## 初期化シーケンス

ページ読み込み時に以下の順序で実行されます。

1. `updateDisplay()` — タイマー表示の初期化
2. `loadTodayCount()` — 今日・累計のセッション数を取得
3. `initGameUI()` — ゲーミフィケーション UI の初期化（A/B: on 時のみ）
4. `Notification.requestPermission()` — ブラウザ通知の権限リクエスト
