# アーキテクチャ概要

## 全体構成

ポモドーロタイマーアプリはシングルファイルの Flask アプリケーションです。

```
1.pomodoro/
├── app.py            # Flask アプリ本体（全ロジックを含む）
├── requirements.txt  # 依存パッケージ（flask>=3.0）
├── pomodoro.db       # SQLite データベース（実行時に自動生成）
└── templates/
    └── index.html    # タイマー画面（CSS・JavaScript を内包）
```

## レイヤー構成

アーキテクチャはレイヤー分離を行わず、`app.py` に全機能を集約しています。

```
┌──────────────────────────────────┐
│   テンプレート (templates/)       │  HTML + CSS + JavaScript (Chart.js)
├──────────────────────────────────┤
│   ルート / ビュー (app.py)        │  Flask ルートハンドラ
├──────────────────────────────────┤
│   ビジネスロジック (app.py)       │  統計計算・バッジ評価・XP/レベル計算
├──────────────────────────────────┤
│   データアクセス (app.py)         │  SQLite（sqlite3 モジュール直接使用）
└──────────────────────────────────┘
```

## 主要コンポーネント

### Flask アプリ設定

- `app.secret_key`: 環境変数 `SECRET_KEY` または固定フォールバック値
- データベースパス: `app.py` と同じディレクトリの `pomodoro.db`
- 起動ポート: 環境変数 `PORT`（デフォルト: 5000）
- デバッグモード: 環境変数 `FLASK_DEBUG=true/1/yes` で有効

### ユーザー識別

Cookie ベースの Flask セッションにより匿名ユーザーを識別します。初回アクセス時に UUID を生成し、セッションクッキー (`user_id`) に保存します。ログイン機構はありません。

### A/B テスト機構

ゲーミフィケーション機能の有効・無効を A/B テストで制御します。グループ判定の優先順位は以下の通りです。

1. クエリパラメータ `?ab=on/off`（テスト用オーバーライド）
2. 環境変数 `GAMIFICATION_ENABLED`（サーバー全体の強制設定）
3. `ab_assignments` テーブルの永続化済み割り当て
4. ランダム割り当て（50/50）

### XP / レベルシステム

| 定数               | 値    |
|------------------|-------|
| `XP_PER_POMODORO` | 100  |

レベルは 1〜10 で、必要累計 XP の閾値に基づきます。

| レベル | 必要累計 XP |
|------|------------|
| 1    | 0          |
| 2    | 200        |
| 3    | 500        |
| 4    | 1,000      |
| 5    | 2,000      |
| 6    | 3,500      |
| 7    | 5,500      |
| 8    | 8,000      |
| 9    | 11,000     |
| 10   | 15,000     |

### データベース

SQLite を `sqlite3` 標準ライブラリで直接操作します。`g` オブジェクト経由でリクエストごとにコネクションを管理し、リクエスト終了時に `teardown_appcontext` でクローズします。

テーブルは `init_db()` 関数で初期化され、`__main__` ブロックからサーバー起動時に呼び出されます。

### フロントエンド

`templates/index.html` に CSS・JavaScript をインライン記述した単一 HTML ファイルです。外部依存は Chart.js（CDN）のみです。

## 技術スタック

| 要素           | 技術                        |
|--------------|---------------------------|
| Webフレームワーク | Flask 3.x                 |
| テンプレートエンジン | Jinja2（Flask 同梱）       |
| データベース    | SQLite 3（標準ライブラリ）   |
| フロントエンド  | バニラ JavaScript           |
| チャートライブラリ | Chart.js 4（CDN）          |
| Python バージョン | 3.x（型ヒント使用）          |
