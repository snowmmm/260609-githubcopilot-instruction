# ポモドーロタイマー 実装機能一覧

## コア機能（必須）

| # | 機能 | 詳細 |
|---|---|---|
| 1 | **タイマーカウントダウン** | 25:00 から 00:00 に向けて 1 秒ずつカウントダウン |
| 2 | **開始 / 一時停止** | 「開始」ボタンで開始、再押下で一時停止 |
| 3 | **リセット** | 「リセット」ボタンで現在のモードの初期時間に戻す |
| 4 | **モード切り替え** | 作業（25 分）→ 短休憩（5 分）→ 作業… のサイクル管理 |
| 5 | **タイマー完了通知** | カウントダウンが 00:00 になったらブラウザ通知または音で通知 |

## UI 表示（必須）

| # | 機能 | 詳細 |
|---|---|---|
| 6 | **円形プログレスバー** | 残り時間に応じて SVG の弧を動的に更新 |
| 7 | **モード表示** | 「作業中」「短休憩」などを画面上部に表示 |
| 8 | **時間表示** | MM:SS 形式でリアルタイム更新 |
| 9 | **今日の進捗パネル** | 完了数・集中時間（分）を表示 |

## バックエンド / データ永続化（必須）

| # | 機能 | 詳細 |
|---|---|---|
| 10 | **セッション記録** | ポモドーロ 1 回完了時にサーバーへ POST して SQLite に保存 |
| 11 | **今日の統計取得** | 完了数・合計集中時間を API で返す（`GET /api/stats/today`） |
| 12 | **DB 初期化** | アプリ起動時に `sessions` テーブルが存在しなければ自動作成 |

## 拡張機能（あると良い）

| # | 機能 | 詳細 |
|---|---|---|
| 13 | **ロングブレイク** | 4 ポモドーロごとに長休憩（15 分）へ自動切替 |
| 14 | **タイトルバー更新** | ブラウザタブのタイトルを残り時間で更新 |
| 15 | **設定変更** | 作業時間・休憩時間をカスタマイズ可能にする |
| 16 | **過去の履歴表示** | 日別の完了数・集中時間グラフ |

## 実装優先順位

```
フェーズ 1（最小動作）: #1, #2, #3, #6, #7, #8
フェーズ 2（データ保存）: #4, #5, #9, #10, #11, #12
フェーズ 3（拡張）    : #13, #14, #15, #16
```
